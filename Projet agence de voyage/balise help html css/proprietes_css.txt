PROPRIÉTÉS CSS ESSENTIELLES

Sélecteurs :
* : Sélecteur universel.
.element : Classe (sélecteur par .nom).
#id : Sélecteur par identifiant.
balise : Sélecteur par balise (ex: div, p, h1).

Mise en page :
display : Type d’affichage (block, inline, flex, grid...).
position : Positionnement (static, relative, absolute, fixed, sticky).
top / right / bottom / left : Positionnement relatif/absolu.
z-index : Ordre d’empilement des éléments.

Flexbox :
display: flex : Active Flexbox.
justify-content : Alignement horizontal (flex-start, center, space-between...).
align-items : Alignement vertical (stretch, center...).
flex-direction : Direction des éléments (row, column).
flex-wrap : Gestion du retour à la ligne.

Grid :
display: grid : Active Grid.
grid-template-columns / rows : Définit la grille.
gap : Espacement entre les cellules.

Texte :
color : Couleur du texte.
font-size : Taille du texte.
font-family : Police du texte.
font-weight : Épaisseur du texte (normal, bold, 100-900).
text-align : Alignement du texte (left, center, right).
line-height : Hauteur de ligne.
letter-spacing : Espacement des lettres.
text-decoration : Soulignement, barré, etc.

Couleurs et fonds :
background-color : Couleur d’arrière-plan.
background-image : Image de fond.
background-size : Taille de l’image de fond (cover, contain).
background-position : Position de l’image de fond.

Tailles et boîtes :
width / height : Largeur et hauteur.
max-width / min-width : Contraintes de largeur.
margin : Marges externes.
padding : Marges internes.
border : Bordure (épaisseur, style, couleur).
border-radius : Coins arrondis.
box-shadow : Ombre portée.

Effets :
opacity : Transparence (0 à 1).
transition : Animation des changements CSS.
transform : Transformations (rotate, scale, translate...).
animation : Animations avec @keyframes.

Listes et tableaux :
list-style : Style des puces (none, disc, square...).
border-collapse : Fusion des bordures de tableau.

Divers :
cursor : Apparence du curseur (pointer, default...).
overflow : Gestion du dépassement (hidden, scroll, auto).
